    3つの主要ウェブサイトを一つのアプリに！

    Qiita、スタックオーバーフロー、Teratailで気になることを調べよう。

    Search by Tag
    タグで検索

    Make your favorite tags List. You can explore tags, subscribe them and save as favorite tags. Once it is subscribed, you can access without searching them again.

    お気に入りのタグリストを作ってカスタマイズしよう。１タッチで簡単にタグを購読しよう！

    Keyword Search
    キーワード検索

    Search more specific information using keywords. This app will save the history. This mode is only available for Qiita and Stack Overflow.

    キーワード検索にも対応。(Qiita, スタックオーバーフロー) 履歴検索にも対応。
